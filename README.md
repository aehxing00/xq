# 中国象棋 AI 版

这是一个基于 HTML5、CSS3 和 JavaScript 开发的高级中国象棋游戏。

## 功能特点

*   **高级 AI**: 内置 Alpha-Beta 剪枝算法和位置估值表 (PST)，具有一定棋力。
*   **精美 UI**: 采用 SVG 矢量绘图绘制棋盘，支持高分屏，具有木纹质感和拟物化棋子。
*   **游戏功能**: 支持悔棋、重新开始、难度选择（搜索深度）。
*   **规则完善**: 实现了飞将（将帅照面）检测、蹩马腿、塞象眼等基础行棋规则。

## 如何运行

1.  直接用浏览器打开 `index.html` 文件即可。
2.  推荐使用 Chrome, Edge 或 Firefox 等现代浏览器。

## 开发说明

*   `index.html`: 页面结构。
*   `style.css`: 样式表，包含 SVG 背景定义。
*   `script.js`: 游戏核心引擎，包含走法生成、规则判定和 Minimax AI。
